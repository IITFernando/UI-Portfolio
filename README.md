# UI / UX Portfolio — Fernando

Este repositorio reúne proyectos de diseño UI/UX enfocados en claridad, usabilidad y criterio visual.
Mi interés principal está en diseñar interfaces simples, funcionales y coherentes, pensadas para personas reales y contextos reales.

## Enfoque
Trabajo desde el problema hacia la solución, priorizando:
- comprensión del contexto de uso
- reducción de fricción
- consistencia visual
- decisiones de diseño fundamentadas

Cada proyecto documenta el proceso completo: idea, exploración, diseño y resultado final.

## Proceso
Según el proyecto, aplico:
- Research y definición del problema
- Wireframes y arquitectura de información
- Diseño UI
- Prototipos interactivos

## Herramientas
- Figma  
- Design Thinking  
- UI Systems  
- Prototyping
